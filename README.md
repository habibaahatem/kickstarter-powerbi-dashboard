# kickstarter-powerbi-dashboard
Kickstarter Projects Analysis – Power BI

Designed and developed an end-to-end Power BI dashboard to analyze Kickstarter projects using datasets from 2016 and 2018.

Performed data cleaning, transformation, and integration using Power Query (M Language).

Appended multiple datasets and created a unified data model.

Built a project hierarchy (Main Category → Category → Project Name) to enable drill-down analysis.

Developed DAX measures to calculate:

Number of Projects

Number of Backers

Total Pledged Amount

Total Funding Goal

Designed interactive visuals to analyze project distribution by category, country, state, launch date, and currency.

Delivered actionable insights on crowdfunding trends and funding performance.
